## 执行结果

### 安装的依赖
- `tailwindcss-animate` - 动画支持
- `class-variance-authority` - 变体管理
- `clsx` - 类名合并
- `tailwind-merge` - Tailwind 类名合并
- `lucide-react` - 图标库
- `@radix-ui/react-slot` - Radix UI Slot

### 创建的文件
- `components.json` - shadcn/ui 配置
- `src/renderer/src/lib/utils.ts` - cn 工具函数
- `src/renderer/src/components/ui/button.tsx` - Button 组件（示例）

### 修改的文件
- `tailwind.config.js` - 添加 CSS 变量和动画插件
- `tsconfig.web.json` - 添加 `@/` 路径别名
- `electron.vite.config.ts` - 添加 `@` 别名
- `src/renderer/src/assets/base.css` - 添加 shadcn/ui CSS 变量

### 目录结构
```
src/renderer/src/
├── components/
│   └── ui/          # shadcn/ui 组件
├── lib/
│   └── utils.ts     # cn 工具函数
└── hooks/           # 自定义 hooks
```

### 使用方式

**添加组件：**
```bash
npx shadcn@canary add button
npx shadcn@canary add card
npx shadcn@canary add input
```

**使用组件：**
```tsx
import { Button } from '@/components/ui/button'

<Button variant="default">Click me</Button>
<Button variant="destructive">Delete</Button>
<Button variant="outline">Outline</Button>
```
