# 智能链接识别与剪贴板检测功能

## 任务描述
实现添加用户时智能识别链接类型（用户链接/作品链接），并自动检测剪贴板内容。

## 实现细节

### 链接识别逻辑
```
输入 URL
    │
    ▼
尝试 getSecUserId() → 成功 → 用户链接
    │
    失败
    ▼
尝试 getAwemeId() → 成功 → 作品链接
    │
    失败
    ▼
返回 unknown
```

### 修改文件
1. `src/main/services/douyin.ts`

## 功能说明
- **用户主页链接**: `https://www.douyin.com/user/xxx` 或 `https://v.douyin.com/xxx`
- **作品链接**: `https://v.douyin.com/xxx`（自动提取作者信息）
- 剪贴板检测：Dialog 打开时自动识别并填充链接
