# v1.3.0 Release

## New Features
- **Dashboard**: Data overview page with 4 stat cards, download trend chart, user distribution chart, top tags chart, content level distribution chart (Recharts)
- **Parallel Sync**: Users can now sync multiple users simultaneously instead of one at a time
- **In-App Browser**: "Open Homepage" in user list now opens in app browser (reuses login session) instead of system browser
- **Error Boundary**: Global React error boundary catches crashes and shows retry UI

## Performance
- **getTopTags Optimization**: Replaced JS-side JSON.parse loop with SQLite `json_each()` for tag aggregation
- **Posts Table Indexes**: Added 5 indexes (user_id, sec_uid, create_time, analyzed_at, downloaded_at)

## Fixes
- **Chart Focus Border**: Fixed persistent orange focus border on Recharts chart elements using high-specificity CSS selectors
- **UI Components**: Refined button, badge, slider components for consistency

## Changes
- Added `recharts` dependency
- Updated `.gitignore` to exclude build cache and download data
