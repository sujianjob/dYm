为项目引入 Playwright 端到端测试框架。

## 执行结果

### 安装的依赖
- `@playwright/test` - Playwright 测试框架

### 创建的文件
- `playwright.config.ts` - Playwright 配置
- `e2e/example.spec.ts` - 示例测试

### 修改的文件
- `package.json` - 添加测试脚本
- `.gitignore` - 忽略 Playwright 输出目录

### 可用脚本
```bash
npm run test:e2e        # 运行 E2E 测试
npm run test:e2e:ui     # 打开 Playwright UI 模式
npm run test:e2e:report # 查看测试报告
```

### 配置的浏览器
- Chromium
- Firefox
- WebKit (Safari)

### 注意事项
首次使用需运行 `npx playwright install` 安装浏览器（已在后台执行）。
